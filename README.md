An emotion-driven CLI-based Movie Recommendation System built using Python and MySQL.  
The system allows users to get movie suggestions based on their mood and enables admin-level CRUD operations for managing the movie database.

---

## Features

- Admin authentication system
- Add new movies to database
- Update existing movie records
- Delete movies
- View complete movie database
- Emotion-based movie recommendations
- IMDb redirection using Python's webbrowser module
- MySQL database integration

---

## Tech Stack

- Python
- MySQL
- SQL Queries (CRUD operations)
- Command Line Interface (CLI)

---

## How It Works

1. User selects a mood/emotion.
2. System filters movies based on mapped genre.
3. Movies are fetched from the MySQL database.
4. Recommendations are displayed in the CLI. 
5. Optional IMDb redirection for more details.
6. Admin users can manage movie records securely.
